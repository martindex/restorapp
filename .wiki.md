# Project Summary
The project is a Point of Sale (POS) system designed for restaurants, named "VARES". It streamlines the management of customer orders, reservations, and payments, enhancing operational efficiency and customer experience. The system provides a user-friendly interface for waitstaff, cooks, and cashiers, ensuring smooth interactions in a busy restaurant environment.

# Project Module Description
The project consists of several functional modules:
- **User Management**: Handles user roles and permissions (e.g., administrators, waitstaff).
- **Table Management**: Manages the status and arrangement of tables within the restaurant.
- **Reservation System**: Facilitates customer reservations and tracks availability.
- **Order Processing**: Manages customer orders, including item tracking and payment processing.
- **Audit Log**: Records system changes and transactions for accountability and review.
